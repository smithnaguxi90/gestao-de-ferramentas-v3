## 🎯 Como Usar

### **1. Instalar Dependências**

```bash
npm install
```

### **2. Verificar Código**

```bash
npm run lint          # Verificar erros
npm run format        # Formatar código
```

### **3. Usar Novos Módulos**

```javascript
// Exemplo simples
import { notifications } from './core/NotificationManager.js';
import { metrics } from './core/MetricsManager.js';
import { cacheManager } from './core/CacheManager.js';

// Notificação profissional
notifications.success('Operação realizada!');

// Métricas
metrics.trackAction('tools', 'create');

// Cache inteligente
const data = await cacheManager.getOrSet('key', async () => {
  return await fetchFromDatabase();
}, { ttl: 300000 });
```
